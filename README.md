# Heroes Backend

## Requirements

  - Node v8+
  - MongoDB v3.6.3
  
## Installation

  - npm install
  - ensure MongoDB is running at the 27017 port

## Development server

  - node server.js
  - Run node server.js for a dev server. The API will listen to http://localhost:8080


## API

- GET /api/heroes
- POST /api/heroes
- GET /api/heroes/:id
- PUT /api/heroes/:id
- DELETE /api/heroes/:id
